Pasos para instalar el proyecto

Paso 1 
- instalar PHP VS17 x64 
- Descomprimir PHP en una carpeta (idealmente en una carpeta en c:/)
- Añadir PHP al PATH: en Windows se accede al panel de control - sistema - conficuración avanzada - variables de entorno - PATH y por ultimo agregar la ubicación de la carpeta creada por ejemplo "c:/php"
- verivicar siempre la instalación abriendo el cmd y poniendo el comando php -v

Paso 2
- Instalar PostgreSQL 
- Crear un servidor con el usuario y contraseña por defecto
- Crear la base de datos con nombre "productos_db" para no tener problemas de incompatibilidad de credenciales (En caso de tener una contraseña o usuario diferente ir a conexion_bd.php y cambiar las credenciales)
- Crear las tablas mediante el script dejado en el proyecto, esto copiando el script del archivo bd_productos.sql para luego irnos a pgAdmin4 dar click derecho a la base de datos y seleccionando la opción Query_tool para luego pegar el script y ejecutar.

Paso 3
- Verificar que PHP pueda usar la base de datos, esto abriendo el cmd y poniendo -c C:\php\php.ini-development -m.
- En caso de no aparecer ir a la carpeta raíz de php, buscar php.ini-development abrirlo y quitar la ; a esta dos opciones ;extension=pgsql ;extension=pdo_pgsql para asi habilitar las extensiones.
- Luego reiniciar el servidor php con comando php -S localhost:8000 y luego ctrl + c.
- Verificar nuevamente con el comando -c C:\php\php.ini-development -m y buscar que esten esas extensiones en la lista.

Paso 4
- Abrir el servidor con php en la cmd
- ejecutamos el comando cd "ubicación de la carpeta de proyecto" ej: cd C:\Users\Bastian\Desktop\Registro_productos
- Luego ejecutamos el comando php -c C:\php\php.ini-development -S localhost:8000 que abrirá un servidor localhost de php que alojará la pagina web 
- Abrir la pagina pegando en el navegador http://localhost:8000/Formulario.html
